# WhatsApp Clone Backend

## Description

WhatsApp Clone with:

### Techs

- Socket
- REST
- Swagger
- MongoDB
- AWS
- Docker

## Setup

### Installation

```bash
npm install
```

### Environment

Create a file `.env`, then paste `.env.example` value inside (only for development)

## Running the app

```bash
# development
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
